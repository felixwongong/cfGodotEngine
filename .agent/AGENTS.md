# AGENTS.md

## Agent context

Detailed, always-current agent context lives in **[`.agent/`](.agent/README.md)**.
Read its [`README.md`](.agent/README.md) first each session.

Keep `.agent/` in sync with the project. When the codebase changes, update the
relevant `.agent/` doc and append one line to [`.agent/CHANGELOG.md`](.agent/CHANGELOG.md)
if it exists.

CatSweeper-level usage notes are in CatSweeper's `.agent/systems/subtrees.md`.
